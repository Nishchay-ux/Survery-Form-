# Survey Form

## Description
This is a simple survey form built as part of the FreeCodeCamp curriculum.  
The goal of the project was to practice basic web development concepts by creating a functional and structured form.

## Features
- Input fields (text, email, number)
- Radio buttons and checkboxes
- Dropdown selection
- Form validation using HTML attributes
- Clean and structured layout

## Tech Stack
- HTML5
- CSS3

## What I Learned
- Structuring forms using semantic HTML
- Using different input types effectively
- Basic form validation techniques
- Styling forms with CSS
- Improving layout and spacing
